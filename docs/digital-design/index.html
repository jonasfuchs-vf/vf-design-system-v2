Digital Design

#About the Digital Design Guideline

The following guideline defines the design elements specifying the visual language of the consumer Vodafone website. It also provides guidance on the use of digital assets such as icons, fonts and colour palettes e.a..

The objective is to establish a common web and mobile web experience and driving brand consistency.

Each section describes:

  • Existing variants of an element
  • Principles on how to use them
  • Features and restrictions
  • Rules and do’s & don’ts showing what works well, what doesn‘t and why

A downloadable template is provided for each element (PSD or Sketch).

#How to use our guidelines:

About the different sections:

#Design Principles

provides an overview of the User Experience and User Interface Design principles of the consumer Vodafone website and the responsive design framework.

#Basics

This section guides you through generic design rules like colour usage, borders, shadows or fonts in the digital design.

#Global elements

explains the standard website navigation and footer that should be used on all pages.

#Interaction Elements

describes the existing types of buttons (call to actions) and links.

#Promo Elements

shows the design rules for bubbles and badges.

#UI Components

shows the variety of UI patterns you can use for different purposes.

#WS2 Components

We differentiate between “WS1 Components” and “WS2 Components”. The WS2 Components reflect the Vodafone Group Web Simplification 2.0 styleguide. We carry out incremental changes to the style of the website. Therefore, on some pages you may still find the WS1 style together with WS2 style. When you build a new design (landing page, asset), always prefer to use the WS2 components first.

#Basis Kit UX Design

This Sketch File includes all UI Components we currently have in production in one file. You can use it as workfile for your designs. Each component is described in the dedicated section.
