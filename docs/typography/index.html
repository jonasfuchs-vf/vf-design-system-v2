Typography

#How we use typography

Typography can help create clear hierarchies, organize information and guide users through a product experience.

Layouts may have several levels of architecture or areas that require varying typographic hierarchies. Selecting the appropriate type style is determined by layout or template structure. Text styles are pre-set configurations of our specified Design Tokens such typographic elements as font size, weight, or line height that are specifically calibrated for use on our website.

#Font-Size

Text should be arranged to enable the user to find content of interest easily and quickly.

#Font-Weight

Weights are critical to add emphasis and establish the hierarchy of a page.

#Font-Color

Important information is generally relayed through text. We need to have sufficient color contrast for the text to be legible.
