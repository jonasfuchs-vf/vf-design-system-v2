Typography

#Typography can help create clear hierarchies, organize information, and guide users through a product or experience.

Layouts may have several levels of architecture or areas that require varying typographic hierarchies. Selecting the appropriate type style is determined by layout or template structure.Test styles are pre-set configurations of our specified Design Tokens such typographic elements as font size, weight, or line height that are specifically calibrated for use on our website.

Font size

Text should be arranged to enable the user to find content of interest easily and quickly. This Design Token specifies the size of a font. Changing the font size can affect the size of other parts.

Line hight

Leading, as itâs referred to in print, is a key factor in the rhythm in typography. This Design Token is commonly used to set the distance between lines of text.

Font weight

Weights are critical to add emphasis and establish the hierarchy of a page. This Design Token defines how bold you text are and we decide between three possible values.

Font color

Important information is generally relayed through text. We need to have sufficient color contrast for the text to be legible.

Important information is generally relayed through text

We provide details about Vodafone products and services, and many pages have navigation elements and accordions to allow the user to control when content is displayed to them. We need to have sufficient color contrast for the text to be legible. Therefore, we recommend that all text which acts as a label for a navigation element or provides the user with key information about a product or service should allow sufficient contrast between the colour it is presented in and its background.

A Design Token becomes a style

  • A font-size
  • B line-height
  • C font-weight

The predefined text styles should be used if possible. In some cases, however, it is necessary to define your own definition of font-size, line-height, font-weight and color. Only our defined values are used and combined with each other:

font-size + line-height + font-weight + font-color = text style

We use different values to calculate a line-height (Decimal numbers are rounded). These are multipliers of the font-size:

font-size * value = line-height

$font-size-50 (14px) * $line-height-50 (1.25) = 17,5px = 18px
