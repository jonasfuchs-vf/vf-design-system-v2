Spacings

#The empty space

Every part of a UI should be intentional including the empty space between elements.

The spacing can be applied to margin or padding properties. Likewise, these spacing tokens can be applied to both vertical and horizontal edges. The token takes the place of the values normally assigned to margin and padding. Spacing is flexible based on viewport size.

#Padding

Padding refers to the internal distance of the element, the space between the contents of an element and the element boundary. Padding is used to create distance between content and border of an element.

#Margin

Margin is the outer distance, the space outside the border of an element and is used to position an element. In this way, the distance to other elements or to the side edges can be determined. Margin is not part of an element where padding is part of element.

#Position coordinates

Absolute positioning removes elements from the normal text flow and allows them to be positioned absolutely. Values for left or right, as well as top or bottom can be specified to determine the position. You may also use the property value absoulte for the position of an element: left: A px oder right: B px.

#Spacing Rules

Spacing is an important and often overlooked part of visual design.
