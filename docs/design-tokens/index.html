Design Tokens

#What are Design Tokens?

Design what? Gain design consistency!

Design tokens are pieces of data that take the place of hard-coded properties.

Within the Vodafone Design System, they are the consistent colors, fonts, sizing, and spacing and much more to create a design consistency â the Vodafone Digital Design. Design tokens are defined by variables which can be used by designers and developers to have a common understanding of stylings. By combining Design Tokens we create our Vodafone Digital Style.

  • By using the Design Tokens we will gain a consistent look for our Vodafone products.
  • Design Tokens help us to easily create a consistent design, no matter which flow or page.
  • It is like a common language designers and developers can use to have a same understanding of how styles should look like.

#How do I use Design Tokens?

A Design Token by itself is just a variable that can be combined to create the Vodafone Digital Style.

Use the Design Token variable instead of using specific pixels or hex codes
